## 1. Google Search Console Setup

### Accessing Performance Reports
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Select property: `tesladiyrepair.com`
3. Navigate to **Performance** in the left sidebar

### Key Metrics to Track

| Metric | Description | Target |
|--------|-------------|--------|
| **Impressions** | How many times pages appeared in search results | Growth trend |
| **Clicks** | How many times users clicked through | Growth trend |
| **CTR (Click-Through Rate)** | Clicks / Impressions | > 3% average |
| **Average Position** | Average ranking position | < 10 (page 1) |

### Date Range Recommendations

| Purpose | Date Range | Use Case |
|---------|------------|----------|
| Weekly monitoring | Last 28 days | Track recent trends |
| Monthly reporting | Last 3 months | Compare month-over-month |
| Seasonal analysis | Last 6 months | Identify patterns |
| Year-over-year | Last 12 months | Annual comparison |

### Performance Report Filters
- **Search Type:** Web (default)
- **Date:** Select appropriate range
- **Queries:** See what keywords drive traffic
- **Pages:** See which articles perform best
- **Countries:** Filter for EU market focus
- **Devices:** Mobile vs Desktop breakdown

## 2. Weekly Monitoring Checklist

Complete every Monday morning:

### Traffic Overview
- [ ] Check total impressions vs previous week (+/- %)
- [ ] Check total clicks vs previous week (+/- %)
- [ ] Note any significant changes (> 20% variance)

### Query Analysis
- [ ] Review top 10 performing queries
- [ ] Identify 3-5 new ranking queries
- [ ] Flag queries with high impressions but low CTR (< 2%)
- [ ] Note any queries showing position improvement

### Page Performance
- [ ] Review top 10 pages by impressions
- [ ] Identify pages with declining traffic
- [ ] Check pages with position 11-20 (close to page 1)
- [ ] Note any pages showing significant position changes

### Technical Health
- [ ] Check Index > Pages report for new issues
- [ ] Review any crawl errors
- [ ] Verify sitemap is being crawled
- [ ] Check Core Web Vitals status

### Quick Actions
- [ ] Document any pages needing immediate attention
- [ ] Update optimization priorities if needed
- [ ] Note content gaps from query data

## 4. Identifying Optimization Opportunities

### Category A: High Impression, Low CTR (Position 1-10)

**Symptoms:**
- Ranking on page 1 (position 1-10)
- Many impressions (>100/month)
- Low CTR (< 3%)

**Diagnosis:**
- Title not compelling
- Meta description not enticing
- Rich snippets missing

**Actions:**
1. Rewrite title tag with action words
2. Add numbers to title (e.g., "5 Steps", "2024 Guide")
3. Improve meta description with benefits
4. Add FAQ schema for rich snippets

**Priority:** HIGH - Quick wins with immediate impact

---

### Category B: Position 11-20 (Page 2)

**Symptoms:**
- Ranking on page 2 (position 11-20)
- Good impressions but few clicks
- Content is competitive

**Diagnosis:**
- Content needs more depth
- Missing internal link support
- Needs freshness signals

**Actions:**
1. Add 300-500 more words
2. Add more internal links TO this page
3. Add new sections (FAQ, troubleshooting)
4. Update date and add fresh content
5. Improve word count if below 1500

**Priority:** HIGH - Close to page 1

---

### Category C: High Impressions, Position 20+

**Symptoms:**
- Many impressions (topic has demand)
- Poor position (20+)
- Not competitive yet

**Diagnosis:**
- Content quality below competitors
- Missing important subtopics
- Need more backlinks

**Actions:**
1. Analyze top 3 competitors
2. Identify content gaps
3. Major content rewrite (2x length)
4. Add unique value (videos, diagrams)
5. Build internal link clusters

**Priority:** MEDIUM - Longer-term project

---

### Category D: Declining Traffic

**Symptoms:**
- Position dropping month-over-month
- Clicks decreasing
- Impressions stable or declining

**Diagnosis:**
- Content becoming outdated
- New competitors ranking
- Algorithm update impact

**Actions:**
1. Check if information is still accurate
2. Update all dates and model years
3. Add new sections for latest Tesla models
4. Refresh images if outdated
5. Review and improve internal linking

**Priority:** HIGH - Prevent further decline

---

### Category E: Underperforming New Content

**Symptoms:**
- Published > 3 months ago
- Few or no impressions
- Not indexed or poor position

**Diagnosis:**
- Topic may have low demand
- Content not matching search intent
- Technical indexing issues

**Actions:**
1. Verify page is indexed (site:url search)
2. Check Search Console coverage
3. Review title/meta for keyword alignment
4. Add internal links from established pages
5. Consider consolidating with related content

**Priority:** MEDIUM - Evaluate ROI

## Quick Reference

### Search Console Navigation
- **Performance:** Main metrics dashboard
- **URL Inspection:** Check individual page status
- **Index > Pages:** See indexing status
- **Experience > Core Web Vitals:** Performance metrics
- **Sitemaps:** Verify sitemap submission
- **Links:** See backlinks and internal links

### Ideal Metrics (Goals)
- Average CTR: > 3%
- Articles on page 1: > 30
- Monthly clicks: > 5,000
- Monthly impressions: > 100,000

### Warning Signs
- CTR below 1% (title/meta issues)
- Position dropping > 5 spots (content freshness)
- Impressions dropping > 20% (competitive pressure)
- Zero impressions after 3 months (indexing issue)
