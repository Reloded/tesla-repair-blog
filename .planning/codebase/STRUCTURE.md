```
tesla-repair-blog/

```

## Key Locations

| What | Where |
|------|-------|
| Add new blog post | `src/posts/[slug].md` |
| Edit site layout | `src/_includes/base.njk` |
| Edit post layout | `src/_includes/post.njk` |
| Update site metadata | `src/_data/metadata.json` |
| Modify styles | `src/css/style.css` |
| Configure build | `.eleventy.js` |
| View built site | `_site/` |

## File Naming Conventions

- **Posts:** `tesla-[topic].md` (kebab-case, prefixed with "tesla-")
- **Pages:** `[name].njk` (kebab-case)
- **Templates:** `[name].njk` in `_includes/`
- **Data:** `[name].json` in `_data/`

---
