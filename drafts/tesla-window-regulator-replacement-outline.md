### 2. Symptoms of a Bad Window Regulator (150 words)
- Window moves slowly or unevenly
- Grinding/clicking noise when operating window
- Window falls into door
- Window won't go up (but motor still sounds)
- Error messages on screen (rare)

### 4. Step-by-Step: Removing the Door Panel (300 words + photos)
1. Lower the window partially (if possible)
2. Remove screw covers (small pry tool)
3. Remove T30 screws (usually 2-3)
4. Unclip door panel starting from bottom
5. Disconnect door handle cable
6. Disconnect electrical connectors
7. Set door panel aside safely

**Pro tip:** Take photos of connector locations before disconnecting

### 5. Accessing the Window Regulator (200 words + photos)
1. Tape window glass in UP position (painter's tape across door frame)
2. Remove plastic vapor barrier carefully (reuse it)
3. Locate regulator mounting bolts
4. Disconnect window motor electrical connector
5. Loosen glass mounting nuts (DON'T remove yet)

### 6. Removing the Old Regulator (250 words + photos)
1. Support the glass from inside the door
2. Remove glass mounting nuts completely
3. Carefully lift glass out of regulator track
4. Remove regulator mounting bolts (usually 3-4)
5. Maneuver regulator out through door opening
6. Note orientation for reinstall

**Warning:** Glass is heavy and fragile — have a helper if possible

### 7. Installing the New Regulator (250 words + photos)
1. Compare old vs new regulator (confirm match)
2. Feed new regulator into door cavity
3. Hand-start mounting bolts (don't tighten yet)
4. Align regulator tracks with glass position
5. Lower glass carefully into new regulator track
6. Install glass mounting nuts (hand-tight first)
7. Connect motor electrical connector
8. Test window operation before final tightening
9. Tighten all mounting bolts
10. Tighten glass mounting nuts

### 8. Reassembly & Calibration (200 words)
1. Reinstall vapor barrier
2. Reconnect door panel electrical connectors
3. Reconnect door handle cable
4. Snap door panel back into place
5. Install screws and covers
6. **Window calibration procedure:**
   - Close window fully → hold UP for 5 seconds
   - Open window fully → hold DOWN for 5 seconds
   - Close again — should now auto-close properly

### 9. Troubleshooting Common Issues (150 words)
- Window still doesn't work → check motor connector
- Window moves but is crooked → adjust glass mounting nuts
- Auto-close doesn't work → repeat calibration procedure
- Grinding noise persists → check for debris in track
